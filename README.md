# The Story

This is the website inspired by the Awwwards websites, and is going to be worked on to be submitted to Awwwwards in the future. This website is also taking an inspiration from sci-fi novel 1984, and movie 1408 by Stephen King, as well as some minor inspiration from Gravity Falls. The story that the website will be presenting is about AI taking over the humanity. Where the main character is a graphic/web designer, who gets laid off due to being replaced by AI.

## What's done so far

Currently the project introduces the person's inventory, which is the wallet combined with the journal. Where the journal serves as checkpoints/navigation throughout the website and it's story. The time/timer presented at the top left, which is used to share the date and time in the game. And the FAQ menu at the top right, to show the website progression, and what the user is meant to be doing currently.

## Made with

This project is made with Nextjs, with Typescript and Tailwind, also Framer Motion for animations. For the art in the website which was all made by me, I've used Procreate.
